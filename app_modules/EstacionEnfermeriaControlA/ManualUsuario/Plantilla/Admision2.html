MANUAL DE USUARIO SIIS

I. ADMISIONES

1.1. BUSCAR PACIENTE
Aparece una ventana como se muestra el Figura No. Buscar Paciente, con las siguientes opciones:
PLAN:Lista los diferentes tipos de Planes que posee la institución.
TIPO DOCUMENTO:Lista los diferentes tipos de documentos de identificación.
DOCUMENTO:Campor para digitar el número del documento del paciente.

Figura No. Buscar Paciente


NOTA 		El botón MENU permite regresa al anterior, Menu Admision de Urgencias. EL botón BUSCAR es utilizado para ir a buscar un paciente y determinar si este posee un Ingreso y Cuenta activos.

IMPORTANTE 		El campo Documento y Plan son obligatorios y siempre debe estar llenos cuando se va a buscar un paciente. En caso de tratarse de un NN el campo Documento no sera requerido y la identificacion sera asignada automaticamente.

PRECAUCION 		Cuando el paciente tiene un Ingreso y Cuenta activos no se permite la Admisión de este paciente a la institución.

1.1.1.1. AUTORIZACIONES
Después de buscar un paciente se hace una autorización para su ingreso a la institución.

NOTA 		El proceso realizado en Autorizaciones puede ser consultado en el capitulo Autorizaciones.
